Different Types of Selectors

Selectors are used to style HTML elements according to their type and/or attributes.
All markups can be selected with the universal selector: " * "{ /* declarations here */ }

Select by Syntax Example
Type element-name { /* declarations here */} p { text-align: center; color: red;}
Attribute 'id' (in the HTML file) '#' (in CSS file)
'class' (in HTML file) '.' (in CSS file)		 #para1 { text-align: center; color: red; }
.center { text-align: center; color: red;}

CSS Units

Absolute Units Relative Units
  • em: Property size relative to property size of direct parent element (most common)
  • rem: Property size relative to property size of direct root element
  • vw: Percentage based on width of screen
  • vh: Percentage based on height of screen
  • px: Pixels (most common)
  • pt: Points
  • mm: Millimeters
